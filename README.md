# Claude Agent Dashboard

A real-time monitoring dashboard for [Claude Code](https://claude.com/claude-code) agent teams.

## Quick Start

```bash
npm install
npm start
```

Open http://localhost:5173

## Features

- Real-time agent monitoring
- Task tracking and progress
- Inter-agent communication
- Live activity streams
- System metrics
